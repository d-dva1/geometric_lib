# Документация проекта geometric_lib

## Общее описание 
Geometric Library - это библиотека для вычисления площадей и периметров для крука и квадрата.
Библиотека может быть использована для учебных целей или в дальнейшем для реализации сложных вычичлений, для каждной функции присутсвуют комментарии.

## Описание функций

### Круг

**Файл**: `circle.py`

#### `area(r)`
Вычисляет площадь круга по радиусу.
**Формула**
S = pi * r^2

**Пример работы**
```python
area(2) # 12.56637061
```

#### `perimetr(r)`
Вычисляет периметр круга по радиусу.
**Формула**
P = 2 * pi * r

**Пример работы**
```python
perimetr(4) # 25.13274123
```

### Квадрат

-**Файл**: `square.py`

#### `area(a)`
Вычисляет площадь квадрата.
**Формула**
S = a^2

**Пример работы**
```python
area(2) # 4
```

#### `perimetr(a)`
Вычисляет перимтр квадрата.
**Формула**
P = 4 * a

**Пример работы**
```python
area(2) # 8
```
